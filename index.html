DO PROJETO À BUILD

Construa e finalize seu primeiro jogo. Resultado final: build jogável.

O que será construído

Você vai montar um jogo, não colecionar aulas.

O treinamento acompanha a criação de um Space Shooter 2D do primeiro objeto na cena até a build final. Cada módulo existe para colocar uma parte do jogo de pé.

MOD-01

Movimento da nave

Entrada do jogador, limites de tela e controle básico.

MOD-02

Sistema de tiros

Prefabs, cadência, disparos e limpeza dos objetos.

MOD-03

Inimigos

Spawn, movimento simples e pressão sobre o jogador.

MOD-04

Colisões

Contato, dano, destruição e regras centrais do jogo.

MOD-05

Pontuação

Contagem de pontos e resposta clara de progresso.

MOD-06

Interface

HUD, estados de partida e informações úteis na tela.

MOD-07

Som

Efeitos simples para reforçar impacto, tiro e derrota.

MOD-08

Menu

Tela inicial, reinício e fluxo mínimo de jogo.

MOD-09

Build final

Exportação de uma versão jogável para mostrar.

Plano técnico

Plano de treinamento em 4 semanas

SEMANA 01 Fundamentos
  • Unity sem atalhos confusos
  • Organização da cena
  • Movimento do jogador
  • Primeiros scripts em C#
SEMANA 02 Tiros e colisões
  • Tiros e prefabs
  • Inimigos e spawns
  • Colisões e dano
  • Pontuação funcional
SEMANA 03 Interface e feedback
  • HUD de jogo
  • Tela de derrota ou vitória
  • Sons e sinais de resposta
  • Ajustes de ritmo
SEMANA 04 Build e entrega
  • Menu inicial
  • Revisão do projeto
  • Exportação da build
  • Entrega do primeiro jogo

Entrega final

O objetivo é simples: uma build jogável.

O resultado esperado não é virar profissional em quatro semanas. É terminar o primeiro jogo, entender a base e ter um projeto real para continuar evoluindo.

  • Entender a estrutura de um projeto Unity 2D
  • Criar mecânicas simples com C#
  • Organizar objetos, cenas e prefabs
  • Montar HUD, menu e estados de partida
  • Testar, ajustar e exportar uma build
  • Finalizar o primeiro jogo para portfólio
